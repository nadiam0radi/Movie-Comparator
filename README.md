# 🎬 Movie Comparator API

Movie Comparator API is a simple web API that allows users to search for two movies and compare their details, ratings, and awards.

This project is useful for movie comparison websites, portfolio projects, and learning how to work with external movie APIs.

---

## 🚀 Features

- Search for movies by title
- Display detailed movie information
- Compare two movies side by side
- Show ratings from different sources
- Display awards and nominations
- Return clean JSON responses
- Easy to integrate with a frontend application

---

## 📌 Movie Information

For each movie, the API can return:

- Title
- Release year
- Genre
- Director
- Actors
- Plot
- Runtime
- IMDb rating
- Rotten Tomatoes rating
- Metacritic score
- Awards
- Poster image
